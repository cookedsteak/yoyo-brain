# 模型调用策略

### 画面的并行分析
yoyo-eye为摄像头输入端，可以设置该摄像头所需要进行识别的模型。
用'.'分离。例如：'modela.modelb.modelc'。
这样设置后，模型 modela，modelb，modelc 都能够拿到该画面进行并行识别。

### 画面的流式分析
流式分析暂时使用另起进程来进行识别。没有特殊的设置选项。


模型和队列的关系为 1:1。暨一个队列中的数据只能被一个模型消耗。
